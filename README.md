
## INSTALATION

### MODO standar 
Agregamos la carpeta entera con todos los comandos al PATH.

```bash
PATH="$PATH:/rutaDondeTenemosNuestro/gitbash-lf"
export PATH
```
------------------------------------------

### MODO 1  (uno por uno)

Ejemplos:
```bash
ln -s /root/dev-lfrakie/proyectos/gitbash-lf/git-up /usr/bin/git-up
```

O tmbn asi:

```bash
ln -s $PWD/git-up
```
